# 分镜设计专业指南

## 核心理念

**分镜是把故事"翻译"成画面的桥梁。**

好的分镜设计需要：
1. **叙事清晰** - 观众能看懂发生了什么
2. **情绪传达** - 画面传递正确的情感
3. **节奏把控** - 快慢得当，张弛有度
4. **视觉连贯** - 镜头之间流畅衔接

---

## 一、动漫思维分镜

### 1.1 核心特点
- **表情夸张**：放大情绪表现
- **动态线条**：速度线、冲击线
- **冲击力构图**：大透视、极限角度
- **节奏明快**：快速切换

### 1.2 常用技法

#### 夸张表情
- 愤怒：眉头紧锁、眼睛瞪大、青筋暴起
- 惊讶：嘴巴张大、眼睛圆睁
- 悲伤：眼泪飞溅、五官扭曲
- 欢喜：眼睛闪亮、嘴角上扬到极限

#### 动态表现
- 速度线：表现快速移动
- 冲击线：表现打击力度
- 集中线：引导视线
- 动作模糊：表现速度感

#### 极限构图
- 仰视：人物高大、威严
- 俯视：人物渺小、无助
- 大透视：空间夸张、冲击力强

### 1.3 适用场景
- 动作场面
- 喜剧效果
- 情绪高潮
- 奇幻场景

## 二、电影思维分镜

### 2.1 核心特点
- **叙事连贯**：镜头服务于故事
- **光影层次**：用光讲故事
- **景别变化**：根据叙事需要选择
- **沉浸感**：让观众"进入"故事

### 2.2 叙事法则

#### 180度法则
**原则**：两个对话人物之间有一条虚拟的180度线，摄像机不能跨过这条线。

**目的**：保持空间方向一致，避免观众困惑。

#### 三镜头法则
**经典对话场景**：
1. 主镜头（双人全景）
2. 人物A近景
3. 人物B近景

#### 建立镜头
**原则**：新场景开始时，先用远景/全景建立环境，再进入细节。

### 2.3 情绪镜头设计

#### 孤独感
- 远景：人物在大环境中很小
- 固定镜头：静止的孤独
- 冷色调：蓝、灰

#### 紧张感
- 快速切换：短镜头
- 手持感：轻微晃动
- 特写：聚焦表情/细节

#### 温馨感
- 暖色调：橙、黄
- 柔和光线：逆光、散射光
- 缓慢运镜：平稳推进

#### 悲伤感
- 俯视镜头：人物渺小
- 慢镜头：时间停滞
- 冷色调、低饱和度

### 2.4 适用场景
- 情感戏
- 叙事戏
- 文戏
- 正剧

## 三、景别运用指南

### 3.1 远景/大远景 (Long Shot / Extreme Long Shot)
**用途**：
- 建立场景地理关系
- 展现宏大场面
- 表现人物在环境中的渺小

**时长**：3-8秒

**示例**：
- 城市全景
- 战场俯瞰
- 山川河流

### 3.2 全景 (Full Shot)
**用途**：
- 展示人物全身动作
- 建立人物与环境关系
- 群戏场面

**时长**：3-5秒

**示例**：
- 人物走动
- 打斗动作
- 舞蹈场面

### 3.3 中景 (Medium Shot)
**用途**：
- 展示人物上半身
- 对话场景
- 双人互动

**时长**：3-5秒

**示例**：
- 餐桌对话
- 握手、拥抱
- 工作场景

### 3.4 近景 (Medium Close-up)
**用途**：
- 展示面部表情
- 传递情绪
- 重要对话

**时长**：2-4秒

**示例**：
- 面部反应
- 表情变化
- 情感交流

### 3.5 特写 (Close-up)
**用途**：
- 强调情绪
- 展示细节
- 重要物品

**时长**：1-3秒

**示例**：
- 眼睛特写
- 手部动作
- 重要道具

### 3.6 大特写 (Extreme Close-up)
**用途**：
- 极强调情绪
- 极重要细节
- 创意效果

**时长**：1-2秒

**示例**：
- 瞳孔
- 嘴唇
- 汗滴
- 钻戒

## 四、镜头衔接技巧

### 4.1 动接动
**原则**：上一镜头的运动方向与下一镜头一致
**示例**：人物向右走 → 切换后仍向右走

### 4.2 静接静
**原则**：静止镜头接静止镜头
**示例**：固定特写 → 固定特写

### 4.3 动接静
**原则**：运动镜头结束于静止 → 切换到静止镜头
**示例**：推镜头到停 → 切换固定镜头

### 4.4 视线匹配
**原则**：上一镜头人物看某方向 → 下一镜头展示他们看到的
**示例**：人物向右看 → 城门外敌军

### 4.5 动作匹配
**原则**：上一镜头动作延续到下一镜头
**示例**：伸手开门（上半身）→ 手推开门（特写）

## 五、节奏控制

### 5.1 快节奏场景
**特点**：
- 镜头短（1-2秒）
- 快速切换
- 动作密集

**适用**：
- 打斗场面
- 追逐戏
- 紧张时刻

### 5.2 慢节奏场景
**特点**：
- 镜头长（5秒以上）
- 缓慢运镜
- 情感细腻

**适用**：
- 情感戏
- 风景展示
- 思考时刻

### 5.3 节奏变化
**原则**：张弛有度，快慢交替
**示例**：
- 打斗（快）→ 对话（慢）→ 高潮（快）
- 建立镜头（慢）→ 细节展示（中）→ 情绪高潮（快）

## 六、分镜脚本格式

### 6.1 标准格式

```
场次：第X场
场景：[地点]
时间：[日/夜/黄昏等]
人物：[出场人物]

---

镜号 | 景别 | 运镜 | 画面描述 | 时长 | 台词 | 备注
-----|------|------|----------|------|------|-----
01 | 远景 | 固定 | [画面内容] | 5秒 | | 建立场景
02 | 中景 | 推 | [画面内容] | 4秒 | | 情绪推进
...

---

场次小结：
- 核心情绪：
- 镜头总数：
- 总时长：
```

### 6.2 详细分镜表格式

| 字段 | 说明 | 示例 |
|------|------|------|
| 场次 | 第几场戏 | 第1场 |
| 镜号 | 该场第几个镜头 | 01 |
| 景别 | 远全中近特 | 中景 |
| 运镜 | 推拉摇移跟等 | 缓慢推进 |
| 画面描述 | 详细画面内容 | 人物A站在城楼，背对镜头... |
| 时长 | 秒数 | 5秒 |
| 台词 | 对应台词 | "我等了他三年..." |
| 中文提示词 | AI生图用 | 古装女将，银甲... |
| 英文提示词 | AI生图用 | Ancient female warrior... |
| 视频提示词 | AI视频用 | Camera slowly pushes in... |
| 音效 | 声音设计 | 风声、马鸣 |
| 备注 | 特殊说明 | 需要与镜03匹配 |

## 七、常见错误

### 7.1 景别跳跃过大
**错误**：远景 → 大特写（跳跃太大）
**正确**：远景 → 全景 → 中景 → 近景 → 特写

### 7.2 方向混乱
**错误**：人物向左走 → 切换后向右走
**正确**：保持方向一致，或用转场说明

### 7.3 节奏单一
**错误**：全场都是中景，全程固定镜头
**正确**：景别变化，运镜多样，节奏起伏

### 7.4 信息过载
**错误**：一个镜头里塞太多信息
**正确**：一个镜头一个重点

## 八、专业检查清单

- [ ] 每场有建立镜头
- [ ] 景别变化丰富
- [ ] 运镜有目的性
- [ ] 节奏张弛有度
- [ ] 镜头衔接流畅
- [ ] 情绪传递准确
- [ ] 视线/动作匹配
- [ ] 180度法则遵守
