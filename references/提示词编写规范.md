# 📝 提示词编写规范

> 核心理念：**结构完整 + 具体明确 + 中英对照 + 负面约束**

---

## 一、提示词结构

### 1.1 标准结构

```
【主体】→【细节】→【环境】→【光影】→【风格】→【参数】
```

### 1.2 结构详解

#### 主体（Subject）
- 人物：性别、年龄、身份、外貌特征
- 物体：名称、材质、状态
- 场景：地点、类型

#### 细节（Details）
- 人物：服装、表情、动作、发型
- 物体：纹理、图案、装饰
- 场景：元素、布局

#### 环境（Environment）
- 背景：天空、地面、远景
- 氛围：时间、天气、情绪
- 空间：室内/室外、大小、布局

#### 光影（Lighting）
- 类型：自然光/人造光
- 方向：顺光/侧光/逆光
- 色温：暖色/冷色
- 特效：阴影、高光、反射

#### 风格（Style）
- 艺术风格：写实、动漫、油画等
- 时代风格：古代、现代、未来
- 地域风格：中式、日式、欧美

#### 参数（Technical）
- 质量：masterpiece, best quality, 8K
- 视角：front view, side view, low angle
- 画幅：portrait, landscape, square

## 二、人物提示词模板

### 2.1 基础人物提示词

**中文模板：**
```
[角色名]，[性别]，[年龄段]，[身份/职业]。
[脸型]，[眼睛]，[眉毛]，[鼻子]，[嘴唇]。
[发型]，[发色]，[发长]。
[肤色]，[体型]。
[服装描述]，[配饰]。
[表情]，[动作/姿态]。
[气质关键词]。
[风格标签]。
```

**英文模板：**
```
[Name], [gender], [age] years old, [role/profession].
[face shape], [eye description], [eyebrow], [nose], [lips].
[hairstyle], [hair color], [hair length].
[skin tone], [body type].
[clothing description], [accessories].
[expression], [pose/action].
[temperament keywords].
[style tags].
masterpiece, best quality, highly detailed, 8K.
```

## 六、提示词优化技巧

### 6.1 权重调整

```
(word) - 增加权重
(word:1.5) - 1.5倍权重
(word:0.8) - 降低权重
```

### 6.2 否定提示

```
不要出现：not, no, without
例如：no background, no text, without watermark
```

### 6.3 融合技巧

```
使用 "|" 或 "," 连接多个描述
例如：1girl | red hair | blue eyes
```

## 七、常见错误与修正

| 错误 | 问题 | 修正 |
|------|------|------|
| 模糊描述 | 生成结果不明确 | 使用具体词汇 |
| 过长提示词 | AI理解偏差 | 精简核心元素 |
| 矛盾描述 | 生成冲突 | 统一风格和设定 |
| 缺少负面词 | 出现不良元素 | 添加负面提示词 |
| 缺少参数 | 质量不稳定 | 添加质量标签 |
