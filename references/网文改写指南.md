# 📖 网文改写指南

> 适用场景：已有网文章节或小说片段，需要转化为AI可执行的剧本

---

## 核心改写原则

### 四大转换法则

| 原文写法 | 改写方案 | 理由 |
|---------|---------|------|
| 内心独白 | 旁白/台词/表情 | AI难以呈现内心 |
| 宏大场景 | 局部特写 | 细节更可控 |
| 复杂动作 | 静态图+音效 | 降低生成难度 |
| 夸张表情 | 微表情处理 | 更真实自然 |

---

## 一、改写流程

### Step 1：原文分析

**提取要素：**

| 要素 | 提取内容 |
|------|---------|
| 主要人物 | 姓名、身份、特征 |
| 核心场景 | 地点、时间、环境 |
| 主线情节 | 事件、冲突、发展 |
| 情感基调 | 悲伤/欢乐/紧张/温馨 |

**工具：使用[网文改写模板](../templates/网文改写模板.md)**

---

### Step 2：AI可行性预判

**识别高风险元素：**

| 风险类型 | 原文表现 | 改写难度 |
|---------|---------|---------|
| 复杂动作 | 打斗、舞蹈 | 🔴 高 |
| 多人场景 | 3人以上同框 | 🔴 高 |
| 特效场面 | 爆炸、魔法 | 🔴 高 |
| 精确道具 | 文字、图案 | 🟡 中 |

**风险占比 = 高风险镜头数 / 总镜头数**

| 风险占比 | 建议 |
|---------|------|
| <30% | 🟢 可直接改写 |
| 30-50% | 🟡 需优化部分 |
| >50% | 🔴 建议调整剧本 |

---

### Step 3：剧本重构

#### 3.1 镜头拆分原则

```
原文：一句话描述
↓
拆分：多个镜头描述
↓
每个镜头：一句话画面 + 一句台词/旁白
```

**示例：**

| 原文 | 改写后 |
|------|--------|
| "林小雨心里非常矛盾，既想追求梦想，又不想让父母失望" | 【镜1】近景，林小雨低头沉思<br>【镜2】特写，眼眶微红<br>（旁白）："高考倒计时30天..." |

---

#### 3.2 内心戏可视化技巧

**方法1：表情+动作**
| 原文（内心） | 可视化 |
|-------------|--------|
| 心想"这不可能" | 瞪大眼睛，后退一步 |
| 内心挣扎 | 握紧拳头，低头 |
| 感到温暖 | 眼眶湿润，嘴角上扬 |

**方法2：环境映衬**
| 原文（内心） | 环境设计 |
|-------------|---------|
| 心情低落 | 阴天、灰暗色调 |
| 充满希望 | 阳光、明亮色调 |
| 紧张不安 | 手心出汗特写 |

**方法3：旁白补充**
| 原文（内心） | 旁白 |
|-------------|------|
| 回忆过去 | （旁白）："那一年..." |
| 内心独白 | （旁白）："我真的能做到吗？" |
| 思考 | （旁白）："如果...会怎样" |

---

#### 3.3 动作戏简化技巧

**方法1：特写替代**
| 原文（动作） | 改写（特写） |
|-------------|-------------|
| 激烈打斗 | 拔剑特写+刀光音效 |
| 跳舞 | 舞姿静态图+音乐 |
| 奔跑 | 回头特写+脚步音效 |

**方法2：远景虚化**
| 原文（动作） | 改写（远景） |
|-------------|-------------|
| 千军万马 | 战场远景+主将特写 |
| 人群涌动 | 模糊人影+主角特写 |

**方法3：音效替代**
| 原文（动作） | 改写（音效） |
|-------------|-------------|
| 爆炸 | 闪光+爆炸声 |
| 碰撞 | 撞击声+人物反应 |

---

### Step 4：优化检查

**检查清单：**

- [ ] 画面优先：内心描写 → 可视化呈现
- [ ] 场景简化：宏大场景 → 局部特写
- [ ] 动作简化：复杂动作 → 静态图+音效
- [ ] 情感内敛：夸张表情 → 微表情
- [ ] 主角突出：单人镜头占比 > 60%
- [ ] 场景可控：总场景数 ≤ 15个

## 二、改写示例

### 示例1：内心独白改写

**原文：**
> 林小雨心里非常纠结。她从小就喜欢画画，但父母希望她专心学习，考上好大学。这种两难的选择让她整夜难眠。

**改写：**

| 镜号 | 景别 | 画面描述 | 台词/旁白 |
|------|------|---------|-----------|
| 01 | 中景 | 林小雨躺在床上，翻来覆去 | |
| 02 | 特写 | 月光透过窗帘照在她脸上，眼角有泪痕 | |
| 03 | 近景 | 她坐起身，看向墙上的画 | |
| 04 | 特写 | 画上是她的自画像，眼神明亮 | |
| 05 | 中景 | 她又躺下，用手捂住眼睛 | （旁白）："高考倒计时30天，我的梦想...该何去何从" |

---

### 示例2：动作戏改写

**原文：**
> 战斗爆发了！张飞骑在马上挥舞着丈八蛇矛，百万军中取敌将首级，如入无人之境。

**改写：**

| 镜号 | 景别 | 画面描述 | 台词/旁白 |
|------|------|---------|-----------|
| 01 | 特写 | 丈八蛇矛特写，寒光闪闪 | |
| 02 | 远景 | 战场全景，刀光剑影，虚化处理 | |
| 03 | 特写 | 张飞骑马的剪影，披风飘扬 | |
| 04 | 中景 | 敌将惊恐的表情 | |
| 05 | 特写 | 蛇矛刺出 | （音效）：金铁交鸣 |
| 06 | 远景 | 敌将落马 | （旁白）："百万军中，取上将首级" |

---

### 示例3：多人大场景改写

**原文：**
> 教室里坐满了学生，老师在讲台上讲课。

**改写：**

| 镜号 | 景别 | 画面描述 | 台词/旁白 |
|------|------|---------|-----------|
| 01 | 特写 | 林小雨认真听讲的表情 | |
| 02 | 近景 | 她的手在记笔记 | |
| 03 | 中景 | 窗外传来鸟鸣，她微微转头看向窗外 | |
| 04 | 远景 | 教室外景（可用模糊处理） | （音效）：远处老师讲课声 |

## 三、改写技巧总结

### 3.1 镜头语言替代

| 内心表达 | 镜头语言 |
|---------|---------|
| 回忆 | 叠化转场+暖色调 |
| 想象 | 快速切换+特效 |
| 犹豫 | 定格+环境描写 |
| 坚定 | 特写眼神 |

### 3.2 音效氛围替代

| 视觉效果 | 音效替代 |
|---------|---------|
| 爆炸 | 闪光+爆炸声 |
| 风雨 | 人物淋雨特写+雨声 |
| 心情变化 | 环境变化+配乐 |

### 3.3 时间压缩技巧

| 原文（长时间） | 改写（短时间） |
|--------------|--------------|
| 几天几夜 | 几个镜头切换 |
| 长途跋涉 | 出发+到达特写 |
| 慢慢成长 | 关键节点特写 |

---

## 四、注意事项

### 4.1 保持故事完整性
- 改写不能破坏故事主线
- 情感转折要保留
- 关键情节不能遗漏

### 4.2 平衡艺术性与可行性
- 优先保障AI可行性
- 在可行范围内追求艺术性
- 寻找AI特长的表现方式

### 4.3 记录改写原因
- 标注改写点
- 说明改写理由
- 便于后续优化

---

- [ ] 单人镜头占比是否 > 60%？
- [ ] 场景数量是否可控？
- [ ] 改写后情节是否完整？
- [ ] 情感表达是否到位？
