# 🔧 一致性控制方案

> 核心问题：**AI生成的最大痛点：同一人物/场景在不同镜头中外观不一致。**

---

## 一、核心解决方案：风格锚定

### 什么是风格锚定？

风格锚定是指在项目开始前，定义一套统一的视觉风格关键词（风格锚定提示词），并在后续所有生成中重复使用，从而保证画面风格一致性。

### 为什么风格锚定有效？

```
问题根源：
- 每次生成都重新描述风格 → 风格描述不一致 → 画面风格混乱

解决方案：
- 定义一次风格锚定 → 复制粘贴到所有提示词 → 画面风格统一
```

### 风格锚定的组成

一套完整的风格锚定包含：

```markdown
1. 主风格类型（日漫/美漫/CG/仿真人等）
2. 色调偏好（明亮/暗沉/鲜艳/柔和）
3. 画面质感（平涂/厚涂/水彩/油画）
4. 质量关键词（8K/high quality/masterpiece）
5. 负面提示词（排除不需要的风格）
```

## 二、人物一致性控制

### 方案1：风格锚定 + 参考图（推荐）

#### 步骤

1. **完成风格定义**
   - 确定主风格类型
   - 生成风格锚定提示词
   - 准备风格参考图

2. **生成基础人物参考图**
   - 使用「风格锚定 + 人物描述」生成满意的人物图
   - 选择最满意的一张作为标准参考图
   - 存入资产库

3. **后续生成使用参考图**
   - 将参考图作为输入
   - 提示词包含完整的风格锚定
   - 调整场景描述，保持人物特征关键词一致

#### 提示词模板

```
【风格锚定】+ 【人物特征】+ 【新场景描述】

示例：
风格锚定：anime style, high school slice of life, bright and warm colors,
          Makoto Shinkai inspired, clean line art, gentle shading...
人物特征：Lin Xiaoyu, 16 years old, oval face, large bright eyes,
          black shoulder-length hair, white school shirt...
新场景描述：standing by the classroom window, sunlight streaming in...

完整提示词：
anime style, high school slice of life, bright and warm colors,
Makoto Shinkai inspired, clean line art, gentle shading.
Lin Xiaoyu, 16 years old, oval face, large bright eyes,
black shoulder-length hair, white school shirt.
standing by the classroom window, sunlight streaming in.
masterpiece, best quality, 8K.
```

#### 一致性关键词

在所有人物生成中保持一致的核心关键词：

```
【人物核心特征】（必须在所有提示词中出现）
- 姓名/代号
- 年龄
- 核心外貌特征（脸型、发色、发型、眼睛）
- 核心服装特征

示例：
lin xiaoyu, 16 years old, oval face, large bright eyes, tear mole,
black shoulder-length hair, fair skin, gentle
```

### 方案2：三视图设计

#### 步骤
1. 一次性生成人物三视图
2. 分别裁切作为不同角度的参考图
3. 后续生成使用对应角度的参考图

#### 提示词
```
[人物描述], character sheet, three views,
front view, side view, back view,
white background, reference sheet
[风格锚定]
```

#### 使用方法
```
正面镜头 → 使用正面参考图
侧面镜头 → 使用侧面参考图
背面镜头 → 使用背面参考图
```

---

### 方案3：多服装设计

#### 方法

为每个人物设计多套服装，分别生成参考图：

```
人物设计清单：
- 校服版（日常/学校场景）
- 居家服版（家中场景）
- 便装版（外出场景）
- 正装版（特殊场合）
```

#### 提示词模板
```
【风格锚定】+ 【人物特征】+ 【服装描述】

示例：
[风格锚定]
Lin Xiaoyu, 16 years old, oval face, black shoulder-length hair...
wearing pink T-shirt and shorts, casual home wear...
```

## 三、场景一致性控制

### 方案1：风格锚定 + 参考图

#### 步骤
1. 生成场景基础图
2. 作为参考图生成不同角度/时间变体
3. 保持核心元素描述一致

#### 时间变体提示词

```
早晨版：
【风格锚定】+ 【场景基础描述】+ morning light, soft sunlight, fresh atmosphere

黄昏版：
【风格锚定】+ 【场景基础描述】+ sunset lighting, golden hour, warm orange tones

夜晚版：
【风格锚定】+ 【场景基础描述】+ night, moonlight, dark blue tones, street lamps
```

---

### 方案2：分层生成法

#### 方法

```
背景层：固定场景背景
前景层：人物和道具
后期合成
```

#### 优势
- 背景可复用
- 人物独立控制
- 便于后期调整

## 五、一致性检查清单

### 生成前检查

- [ ] 是否已完成风格定义？
- [ ] 是否生成了风格锚定提示词？
- [ ] 是否准备了风格参考图？
- [ ] 核心特征关键词是否明确？
- [ ] 负面提示词是否完善？

### 生成后检查

- [ ] 人物面部是否一致？
- [ ] 发型发色是否一致？
- [ ] 服装是否一致？
- [ ] 整体风格是否统一？

### 问题修复

| 问题 | 解决方案 |
|------|---------|
| 面部不一致 | 检查风格锚定是否完整 + 使用参考图 |
| 发型变化 | 增加发型描述关键词 |
| 服装变形 | 简化服装描述，突出核心特征 |
| 风格差异 | 确保所有提示词都包含相同的风格锚定 |

## 七、常见问题

### Q1: 风格锚定太长怎么办？

**答**：可以精简为核心关键词：

```
完整版（约50词）：
anime style, high school slice of life, bright and warm color palette,
detailed character design, soft lighting with lens flare effect,
fresh and natural atmosphere, high quality Japanese animation,
Makoto Shinkai inspired, clean line art, gentle shading

精简版（约20词）：
anime style, bright warm colors, soft lighting,
Makoto Shinkai inspired, clean line art, masterpiece
```

### Q2: 参考图和风格锚定哪个更重要？

**答**：两者缺一不可：

- **风格锚定**：定义整体视觉风格，保证所有画面风格统一
- **参考图**：保证特定人物/场景的外观一致

### Q3: 不同工具的风格锚定是否通用？

**答**：基本通用，但需要微调：

- GPT-Image-2 / Midjourney / Flux：英文提示词通用
- 部分工具对特定关键词敏感度不同，需要测试调整

## 八、最佳实践

### 1. 先定义风格，再开始设计

```
错误做法：
直接开始画人物 → 发现风格混乱 → 回头修改

正确做法：
定义风格 → 生成风格锚定 → 准备参考图 → 开始人物设计
```

### 2. 风格锚定复制粘贴，不要手写

```
错误做法：
镜头1：anime style...
镜头2：Japanese animation...  ← 描述不一致！

正确做法：
复制粘贴相同的风格锚定到所有提示词
```

### 3. 建立提示词模板库

```
人物提示词模板：
【风格锚定】
【人物核心特征】
【服装/场景描述】
【情绪/动作】
【质量关键词】

场景提示词模板：
【风格锚定】
【场景基础描述】
【时间/天气】
【氛围关键词】
【质量关键词】
```
