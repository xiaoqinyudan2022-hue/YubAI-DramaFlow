# 常见问题与解决方案

## 核心理念

**问题是常态，解决是能力。建立问题库，让每一次失败都有价值。**

---

## 一、静态图常见问题

### 问题1：人物面部变形

#### 表现
- 五官扭曲、不对称
- 眼睛大小不一
- 嘴巴歪斜

#### 原因分析
- 提示词不够精确
- 角度过大（大侧面、仰视）
- 动作幅度过大

#### 解决方案
1. **优化提示词**
   ```
   添加：symmetrical face, balanced features
   负面提示词：asymmetrical, uneven, distorted face
   ```

2. **调整角度**
   - 避免极端角度
   - 使用标准正面/侧面

3. **降低难度**
   - 减少表情幅度
   - 简化头部姿态

---

### 问题2：手部错误

#### 表现
- 多手指（6指、7指）
- 少手指
- 手指变形
- 手掌扭曲

#### 原因分析
- AI对手部理解不足
- 手部位置遮挡/交叉

#### 解决方案
1. **规避方案**
   - 避免手部特写
   - 手部放在身体后面
   - 使用遮挡物

2. **提示词优化**
   ```
   负面提示词：bad hands, extra fingers, missing fingers, deformed hands
   ```

3. **后期修正**
   - 使用PS/美图工具修正
   - 局部重绘

---

### 问题3：肢体畸形

#### 表现
- 关节反向弯曲
- 肢体长度异常
- 多余肢体

#### 原因分析
- 姿势过于复杂
- 肢体交叉/遮挡

#### 解决方案
1. **简化姿势**
   - 避免复杂肢体动作
   - 使用标准站立/坐姿

2. **提示词优化**
   ```
   负面提示词：bad anatomy, extra limbs, missing limbs, floating limbs
   ```

---

### 问题4：背景混乱

#### 表现
- 元素错位
- 透视错误
- 不合理的物体

#### 原因分析
- 场景描述不精确
- AI幻觉

#### 解决方案
1. **详细描述背景**
   - 明确列出背景元素
   - 指定位置关系

2. **分步生成**
   - 先生成干净背景
   - 再生成人物
   - 后期合成

---

### 问题5：风格不一致

#### 表现
- 与参考图风格差异大
- 不同图之间风格漂移
- 画面看起来"拼凑感"强

#### 原因分析
- 风格关键词不一致
- 没有使用风格锚定
- 参考图风格不统一

#### 解决方案
1. **使用风格锚定**
   - 在项目开始前完成风格定义
   - 生成统一的风格锚定提示词
   - 所有提示词都复制粘贴相同的风格锚定

2. **使用参考图**
   - 以满意图片为参考
   - 确保参考图风格一致

3. **检查风格锚定完整性**
   - 每次生成前检查是否包含风格锚定
   - 不要手写风格描述，复制粘贴保证一致性

## 二、视频常见问题

### 问题1：人物变形（视频）

#### 表现
- 面部逐渐扭曲
- 肢体变形
- 身体部位消失/出现

#### 原因分析
- 动作幅度过大
- 时长过长
- 运动复杂

#### 解决方案
1. **缩短时长**
   - 控制在5-8秒
   - 分段生成

2. **简化动作**
   - 只保留必要动作
   - 避免复杂肢体运动

3. **使用图生视频**
   - 先确保静帧质量
   - 再添加简单动态

---

### 问题2：画面闪烁

#### 表现
- 帧与帧之间不连贯
- 色彩跳跃
- 元素突然出现/消失

#### 原因分析
- 提示词不精确
- 生成参数不稳定
- AI理解不一致

#### 解决方案
1. **提高提示词精度**
   - 详细描述每一帧内容
   - 使用精确的时间描述

2. **降低动态幅度**
   - 减少动作变化
   - 缓慢运镜

---

### 问题3：运动不自然

#### 表现
- 动作僵硬
- 违反物理规律
- 速度异常

#### 原因分析
- AI对复杂动作理解不足
- 运动描述不清晰

#### 解决方案
1. **使用自然运动描述**
   ```
   good: "hair flows gently in the wind"
   bad: "hair moves rapidly in multiple directions"
   ```

2. **降低运动速度**
   - 使用"slowly", "gently"
   - 避免快速运动

---

### 问题4：风格漂移

#### 表现
- 视频开始和结束风格不同
- 中途色调变化
- 人物外观渐变

#### 原因分析
- 时长过长
- 动态元素过多

#### 解决方案
1. **控制时长**
   - 单段视频不超过10秒

2. **固定风格词**
   - 在视频提示词中重复风格词

---

## 三、一致性问题

### 问题1：人物不一致

#### 表现
- 同一人物在不同镜头外貌不同
- "变脸"现象

#### 解决方案
1. **建立资产库**（参考一致性控制方案）
2. **使用参考图 + 风格锚定**
3. **固定核心关键词**

### 问题2：场景不一致

#### 表现
- 同一场景在不同镜头差异大

#### 解决方案
1. **生成场景基础图作为参考**
2. **固定场景核心描述词**
3. **所有提示词包含相同的风格锚定**

---

## 四、工具问题

### 问题1：生成速度慢

#### 原因
- 服务器负载高
- 提示词过长
- 分辨率过高

#### 解决方案
- 选择低峰时段生成
- 精简提示词
- 适当降低分辨率

### 问题2：生成失败

#### 原因
- 内容违规
- 参数错误
- 网络问题

#### 解决方案
- 检查提示词是否合规
- 检查参数设置
- 重试或换工具

---

## 五、问题排查流程

```
发现问题
    ↓
是质量问题还是一致性问题？
    ├─ 质量问题 → 评估严重程度
    │    ├─ 严重 → 必须重做
    │    └─ 轻微 → 后期修正
    │
    └─ 一致性问题 → 检查资产库
         ├─ 有资产卡 → 使用资产卡重新生成
         └─ 无资产卡 → 建立资产卡
    ↓
分析原因
    ├─ 提示词问题 → 优化提示词
    ├─ 设计问题 → 简化设计
    └─ 工具问题 → 调整工具使用
    ↓
记录到问题库
```
