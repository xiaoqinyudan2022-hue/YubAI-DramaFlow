# ✅ 五阶段自检制度

> 核心理念：**问题发现越早，修改成本越低。**

---

## 自检成本对比

| 阶段 | 发现问题 | 修改成本 |
|------|----------|----------|
| 故事层 | 最低 | 极低 |
| 风格层 | 低 | 低 |
| 设计层 | 中 | 低 |
| 分镜层 | 高 | 中 |
| 视频层 | 最高 | 高 |

---

## 第一阶段：故事层自检

### 1.1 故事大纲自检

#### 逻辑自检
- [ ] 故事核心冲突是否清晰？
- [ ] 人物动机是否合理？
- [ ] 情节发展是否连贯？
- [ ] 结局是否水到渠成？

#### AI可行性自检
- [ ] 故事中的主要场景是否适合AI生成？
- [ ] 是否有大量复杂动作场面？（打斗、追逐等）
- [ ] 是否有大量人物互动？（多人场景）
- [ ] 是否有难以生成的元素？（精确文字、复杂几何等）

#### 目标受众自检
- [ ] 故事节奏是否适合目标受众？
- [ ] 情感表达是否到位？
- [ ] 主题是否清晰？

#### ⚠️ 高风险信号
- 🔴 故事中有大量战争场面
- 🔴 主要情节依赖精确道具/文字
- 🔴 多人复杂互动过多

**如有高风险信号，建议在故事层就调整设计。**

---

### 1.2 剧本自检

#### 格式自检
- [ ] 场景标题格式是否正确？
- [ ] 人物标注是否清晰？
- [ ] 动作描写是否具体？

#### AI可行性自检
- [ ] 统计总场景数：____ 个
- [ ] 统计高难度场景：____ 个（占比____%）
- [ ] 是否需要简化？（高难度>30%需要调整）

## 第二阶段：风格层自检

### 2.1 风格定义自检

#### 完整性自检
- [ ] 是否已确定主风格类型？（日漫/美漫/CG/仿真人等）
- [ ] 是否已确定色调偏好？（明亮/暗沉/鲜艳/柔和）
- [ ] 是否已确定画面质感？（平涂/厚涂/水彩/油画）

#### 风格锚定自检
- [ ] 是否已生成风格锚定提示词（中文版）？
- [ ] 是否已生成风格锚定提示词（英文版）？
- [ ] 风格锚定是否足够详细？（建议20-50词）
- [ ] 是否已准备负面提示词？

#### 参考图自检
- [ ] 是否已准备至少3张风格参考图？
- [ ] 参考图风格是否与定义一致？

#### ⚠️ 重要提醒
**风格层是保证画面一致性的关键！不要跳过这一步！**

---

### 2.2 风格锚定验证

#### 测试生成
使用风格锚定提示词生成3-5张测试图，检查：

- [ ] 风格是否统一？
- [ ] 色调是否符合预期？
- [ ] 质感是否满意？

#### 如果不理想
- 调整风格锚定关键词
- 更换风格参考图
- 重新测试直到满意

## 第三阶段：设计层自检

### 3.1 人物设计自检

#### 完整性自检
- [ ] 每个主要人物是否有基础设定？
- [ ] 每个主要人物是否有AI提示词？
- [ ] 提示词是否包含风格锚定？
- [ ] 提示词是否中英双语？
- [ ] 是否包含负面提示词？

#### 一致性自检
- [ ] 是否生成并保存了标准参考图？
- [ ] 所有提示词是否都使用了相同的风格锚定？
- [ ] 是否建立了人物资产卡？

#### 难度自检
- [ ] 人物服装是否过于复杂？
- [ ] 是否有AI难以生成的特征？（精细图案、特殊材质）

## 第四阶段：分镜层自检

### 4.1 分镜脚本自检

#### 叙事自检
- [ ] 镜头数量是否足够？（细节越丰富越好）
- [ ] 景别变化是否丰富？（远/中/近/特）
- [ ] 节奏控制是否合理？（动作戏快切，情感戏长镜头）
- [ ] 转场设计是否流畅？

#### AI适配自检
- [ ] 每个镜头的AI难度是否可接受？
- [ ] 是否有高难度镜头的替代方案？
- [ ] 提示词是否完整（中英文+风格锚定）？

---

### 4.2 分镜表自检

#### 完整性自检
- [ ] 每个镜头是否有详细画面描述？
- [ ] 每个镜头是否有中文提示词？
- [ ] 每个镜头是否有英文提示词？
- [ ] 每个镜头的提示词是否包含风格锚定？
- [ ] 每个镜头是否有运镜说明？
- [ ] 每个镜头是否有时长预估？

#### 质量自检
- [ ] 景别比例是否合理？（参考：特写40%、中景30%、全景30%）
- [ ] 运镜设计是否符合AI能力？
- [ ] 动态元素是否可实现？

## 自检流程总结

### 每日自检清单

**故事层结束前：**
- [ ] 故事逻辑检查
- [ ] AI可行性评估
- [ ] 高风险场景调整

**风格层结束前：**
- [ ] 风格定义完整性检查
- [ ] 风格锚定提示词生成
- [ ] 风格参考图准备
- [ ] 测试生成验证

**设计层结束前：**
- [ ] 人物设计检查
- [ ] 场景设计检查
- [ ] 资产库建立
- [ ] 风格锚定一致性检查

**分镜层结束前：**
- [ ] 分镜脚本检查
- [ ] 分镜表检查
- [ ] 提示词完整性检查
- [ ] 风格锚定一致性检查

**视频层结束前：**
- [ ] 静帧图质量检查
- [ ] 视频质量检查
- [ ] 整体成片验收

## 常见问题快速定位

| 问题 | 出现阶段 | 根因阶段 | 解决方向 |
|------|---------|----------|----------|
| 人物不一致 | 视频/分镜 | 设计层 | 检查参考图和风格锚定 |
| 风格混乱 | 视频/分镜 | 风格层 | 重新定义风格锚定 |
| 场景不合理 | 分镜/视频 | 故事层 | 调整剧本场景设计 |
| 动作无法实现 | 视频 | 故事/分镜 | 简化动作或拆分镜头 |
