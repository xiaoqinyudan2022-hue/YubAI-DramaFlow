# 🤝 Contributing Guide

Thank you for your interest in **YubAI-DramaFlow**! We welcome all forms of contributions.

---

## 📋 Ways to Contribute

### 1. Report Issues

If you find a bug or have a feature suggestion:

1. Search [Issues](../../issues) first to avoid duplicates
2. Create a new Issue with a clear title
3. Include:
   - Problem description
   - Steps to reproduce (if it's a bug)
   - Expected behavior
   - Screenshots (if helpful)

### 2. Submit Pull Requests

#### Workflow

1. **Fork this repository**
2. **Create a branch**: `git checkout -b feature/your-feature-name`
3. **Make your changes**
4. **Commit**: `git commit -m 'Add some feature'`
5. **Push**: `git push origin feature/your-feature-name`
6. **Create a Pull Request**

#### Code/Documentation Standards

- Use clear file naming
- Follow Markdown formatting conventions
- Update README when adding new files

### 3. Improve Documentation

We especially welcome:

- 📝 Fixing typos or formatting issues
- 📚 Adding missing documentation
- 💡 Adding more examples to `examples/`

### 4. 🌐 Help with Translations

**We need your help translating this project!**

Current status:
| Language | Status | File |
|----------|--------|------|
| 🇨🇳 Chinese | ✅ Complete | Primary language |
| 🇺🇸 English | 🔄 In Progress | README_EN.md, Quick_Start.md |
| 🇯🇵 Japanese | ❌ Not started | Help needed! |
| 🇰🇷 Korean | ❌ Not started | Help needed! |
| Other languages | ❌ Not started | Help needed! |

**How to help with translations:**

1. Fork the repository
2. Create a translation file (e.g., `README_JA.md` for Japanese)
3. Translate the content
4. Submit a Pull Request

**Translation guidelines:**
- Keep the same structure as the original
- Use natural, native-level language
- Keep technical terms in English when appropriate (e.g., "AI", "Prompt")
- Add yourself to the contributors list

### 5. Share Your Work

If you've created something using this workflow:

- Add your project to `examples/`
- Include complete files (story outline, script, designs)
- Credit yourself as the author
