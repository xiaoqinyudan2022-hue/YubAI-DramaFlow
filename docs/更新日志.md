## [1.0.0] - 2026-04-23

### 🎉 首次发布

**YubAI-DramaFlow（雅布AI漫剧工作流）** 是一套经过实践验证的AI漫剧制作方法论，让AI视频生成从「玄学」变成「可复制的工业化流程」。

### ✨ 核心能力

| 能力模块 | 说明 | 对应文档 |
|----------|------|----------|
| **五阶段流程** | 故事层 → 风格层 → 设计层 → 分镜层 → 视频层 | [五阶段自检制度](../references/五阶段自检制度.md) |
| **风格定义系统** | 解决画面一致性问题的核心方法 | [风格定义模板](../templates/风格定义模板.md) |
| **网文改写模块** | 从小说到大纲到剧本的完整链路 | [网文改写指南](../references/网文改写指南.md) |
| **AI可行性评估** | 提前识别AI难以实现的镜头 | [AI可行性评估模板](../templates/AI可行性评估模板.md) |
| **一致性控制方案** | 风格锚定 + 参考图控制 | [一致性控制方案](../references/一致性控制方案.md) |
| **难度评估体系** | 预判生成难度，优化设计方案 | [AI生成难度评估体系](../references/AI生成难度评估体系.md) |

### 🛠️ 工具推荐

**AI写剧本（御三家）：**
- Claude Opus 4.7 - 指令遵循精准，适合剧本创作
- GPT-6 - 200万Token上下文，适合长篇故事
- Gemini 3.1 Pro - 推理能力强，适合网文改写

**文生图工具：**
- GPT-Image-2（首选）
- NanoBanana系列
- Midjourney
- Flux

**图生视频工具：**
- Seedance 2.0
- 可灵 O3
- Vidu 2.0
- Hailuo
