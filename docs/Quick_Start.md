# 🚀 Quick Start Guide

> Understand the AI drama production workflow in 5 minutes, making AI video generation "replicable" instead of "mysterious"!

---

## 📋 What is AI Drama Production?

**AI Drama Production** refers to using AI tools (text-to-image, image-to-video) to create comic-style short dramas/animations.

### Why Use This Workflow?

| Problem | This Workflow's Solution |
|---------|--------------------------|
| Don't know where to start | 5-stage workflow, each step has a guide |
| Inconsistent character appearance | Style definition + style anchoring |
| Random quality, "gacha" mentality | Standardized templates + self-check system |
| Wasted time on impossible shots | AI difficulty assessment upfront |

---

## 🎯 The 5-Stage Workflow

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  📚 Stage 1: Story Layer                                    │
│     ─────────────────                                       │
│     Story outline → Script → AI feasibility assessment      │
│                                                             │
│  🎨 Stage 2: Style Layer                                    │
│     ─────────────────                                       │
│     Style definition → Style anchoring prompts              │
│                                                             │
│  ✏️  Stage 3: Design Layer                                  │
│     ─────────────────                                       │
│     Character design → Scene design → Asset library         │
│                                                             │
│  🎬  Stage 4: Storyboard Layer                              │
│     ─────────────────                                       │
│     Storyboard script → Shot list → Static frames           │
│                                                             │
│  🎥  Stage 5: Video Layer                                   │
│     ─────────────────                                       │
│     Video prompts → Video generation → Final edit           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 📚 Stage 1: Story Layer

### What to Do

Convert your idea/novel into an AI-friendly script.

### Key Outputs

| Document | Purpose |
|----------|---------|
| Story Outline | Define characters, plot, themes |
| Script | Scene-by-scene breakdown with dialogue |
| AI Feasibility Assessment | Identify hard-to-generate shots |

### Templates to Use

- [Story Outline Template](../templates/故事大纲模板.md)
- [Script Format Template](../templates/剧本格式模板.md)
- [AI Feasibility Assessment](../templates/AI可行性评估模板.md)

### Self-Check Questions

- [ ] Story structure is complete?
- [ ] Characters have clear visual descriptions?
- [ ] Marked potentially difficult AI shots?

---

## 🎨 Stage 2: Style Layer

### What to Do

Define the visual style and create style anchoring prompts.

### Why This Matters

**Style consistency is the #1 challenge in AI video production.** Without proper style anchoring, every shot will look different.

### Key Outputs

| Document | Purpose |
|----------|---------|
| Style Definition | Define overall visual style |
| Style Anchoring Prompts | Keywords to add to every generation |
| Reference Images | Style reference for consistency |

### Templates to Use

- [Style Definition Template](../templates/风格定义模板.md)

### Self-Check Questions

- [ ] Defined core style keywords?
- [ ] Created style anchoring prompts for characters?
- [ ] Collected reference images?

---

## ✏️ Stage 3: Design Layer

### What to Do

Create detailed character and scene designs with AI prompts.

### Key Outputs

| Document | Purpose |
|----------|---------|
| Character Design | Appearance, expressions, prompts for each character |
| Scene Design | Locations, time variants, prompts |
| Asset Library | Organized collection of all designs |

### Templates to Use

- [Character Design Template](../templates/人物设计模板.md)
- [Scene Design Template](../templates/场景设计模板.md)

### Self-Check Questions

- [ ] Each character has multiple angles/expressions?
- [ ] Scenes have time-of-day variants?
- [ ] All prompts tested and working?

---

## 🎬 Stage 4: Storyboard Layer

### What to Do

Create detailed shot list and generate static frames.

### Key Outputs

| Document | Purpose |
|----------|---------|
| Storyboard Script | Shot-by-shot description |
| Shot List | Technical details for each shot |
| Static Frames | AI-generated images for each shot |

### Templates to Use

- [Storyboard Template](../templates/分镜表模板.md)

### Self-Check Questions

- [ ] Each shot has clear camera angle/movement?
- [ ] Static frames match the style?
- [ ] Character consistency checked across shots?

---

## 🎥 Stage 5: Video Layer

### What to Do

Generate videos from static frames and assemble final output.

### Key Outputs

| Document | Purpose |
|----------|---------|
| Video Prompts | Prompts for image-to-video generation |
| Video Clips | Generated video segments |
| Final Edit | Assembled and polished video |

### Self-Check Questions

- [ ] Video prompts include motion description?
- [ ] Generated videos match the style?
- [ ] Transitions smooth between shots?

### For Video Generation

| Tool | Best For |
|------|----------|
