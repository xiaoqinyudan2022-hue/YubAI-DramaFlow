## 🛠️ 技术问题

### Q4: AI生成的人物总是不一致怎么办？

**核心解决方案：**

| 方案 | 适用场景 | 难度 |
|------|---------|------|
| 风格锚定提示词 | 所有项目 | ⭐ |
| 参考图控制 | 需要精确控制 | ⭐⭐ |
| 三视图设计 | 角色固定 | ⭐⭐ |

**推荐流程：**
```
1. 完成风格定义 → 生成风格锚定提示词
2. 生成满意的人物参考图
3. 后续生成时使用参考图 + 风格锚定提示词
4. 保持人物特征关键词一致
```

👉 详见 [风格定义模板](../templates/风格定义模板.md) 和 [一致性控制方案](../references/一致性控制方案.md)

### Q5: 什么镜头AI最难生成？

### 🟢 低难度（成功率 > 80%）
- 单人站立/坐着
- 标准表情（平静、微笑）
- 简单服装
- 自然元素（云、水、头发飘动）

### 🟡 中等难度（成功率 50-80%）
- 双人互动
- 复杂表情（哭泣、愤怒）
- 复杂服装（铠甲、刺绣）
- 特殊光线（逆光、夜景）

### 🔴 高难度（成功率 < 50%）
- 多人场景（3人以上）
- 复杂动作（打斗、舞蹈）
- 精细手部动作（写字、弹琴）
- 特效场面（爆炸、魔法）

👉 详见 [AI生成难度评估体系](../references/AI生成难度评估体系.md)

### Q6: 如何写出高质量的AI提示词？

**标准结构：**
```
【主体】→【细节】→【环境】→【光影】→【风格】→【参数】
```

**示例：**
```markdown
【主体】年轻女性站在窗边
【细节】白色衬衫，马尾辫，侧脸
【环境】教室，阳光透过窗户
【光影】暖色调，自然光
【风格】动漫风格，电影感
【参数】8K，高质量，景深
```

**关键原则：**
1. 具体明确，避免模糊词汇
2. 中英双语，便于调试
3. 添加负面提示词
4. 保持风格关键词一致

👉 详见 [提示词编写规范](../references/提示词编写规范.md)

### Q7: 如何设计适合AI的视频分镜？

**核心原则：**

| 原则 | 说明 |
|------|------|
| 镜头宁可多不可少 | 细节越丰富越好 |
| 景别要有变化 | 远景建场景，近景传情绪 |
| 节奏要控制 | 动作戏快切，情感戏长镜头 |
| 转场要规划 | 提前设计，写在提示词里 |

**推荐景别比例：**
- 特写/近景：40%
- 中景：30%
- 全景/远景：30%

👉 详见 [视频提示词指南](../references/视频提示词指南.md)

## 📋 流程问题

### Q8: 五阶段流程可以跳过吗？

**强烈建议不要跳过！** 每一阶段都是下一阶段的基础。

| 跳过阶段 | 后果 |
|---------|------|
| 跳过故事层 | 剧本质量差，后期大改 |
| 跳过风格层 | 风格混乱，画面不一致 |
| 跳过设计层 | 人物场景不一致，工作量翻倍 |
| 跳过资产层 | AI生成质量不可控 |
| 跳过分镜层 | 镜头混乱，节奏失控 |

---

### Q9: 发现问题已经晚了怎么办？

**五阶段自检的重要性：**

| 阶段 | 发现问题 | 修改成本 |
|------|---------|---------|
| 故事层 | 低 | 极低（改大纲） |
| 风格层 | 低 | 低（改风格定义） |
| 设计层 | 中 | 低（改设计） |
| 分镜层 | 高 | 中（改分镜） |
| 视频层 | 最高 | 高（重新生成） |

👉 详见 [五阶段自检制度](../references/五阶段自检制度.md)

### Q10: 已经有网文/小说，如何改成AI漫剧剧本？

**核心改写原则：**

| 原文 | 改写方案 |
|------|---------|
| 内心独白 | → 旁白/台词/表情 |
| 宏大场景 | → 局部特写 |
| 复杂动作 | → 静态图+音效+旁白 |
| 夸张表情 | → 微表情、细腻处理 |

**改写流程：**
1. 原文分析：提取人物、场景、情节
2. AI可行性预判：识别高风险元素
3. 剧本重构：按剧本格式改写
4. 优化检查：逐项检查适配性

👉 详见 [网文改写指南](../references/网文改写指南.md)

### Q12: 如何选择视频时长？

**推荐单镜头时长：**
- 🟢 3-5秒：固定镜头、微动作（最佳）
- 🟡 5-8秒：简单运镜（可用）
- 🔴 8秒以上：AI控制难度增加

**单集时长建议：**
- 短视频（抖音/快手）：30秒-1分钟
- 中视频（B站）：1-3分钟
- 长视频：3-10分钟
