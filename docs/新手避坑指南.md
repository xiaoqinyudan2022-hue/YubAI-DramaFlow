# 🚨 新手避坑指南

> 这些坑，新手踩过无数次。看完这篇，少走弯路！

---

## 一、故事层常见错误

### ❌ 错误1：剧本设计AI无法实现的场景

**错误案例：**
```
场景：千军万马在战场上厮杀，战马奔腾，刀光剑影
动作：士兵们冲锋陷阵，马匹跳跃，将军挥剑砍杀
```

**问题：**
- 多人同框（AI一致性差）
- 复杂动作（AI做不好）
- 战争场面（AI理解困难）

**正确做法：**
```
场景：主将拔剑立于阵前，远处旌旗飘扬
动作：主将缓缓拔剑，目光坚定望向远方
旁白："唐军将士，随我冲锋！"
```

**原则：** 静态镜头+单人特写+旁白替代复杂动作

---

### ❌ 错误2：人物数量过多

**错误案例：**
```
主要人物：张三、李四、王五、赵六、钱七、孙八...
配角：将军甲、将军乙、侍卫丙...
```

**问题：**
- 6个主要人物 = 6套人物设计 = 一致性噩梦
- 每个镜头都要保证人物不"变脸"

**正确做法：**
```
主要人物：1-2人
配角：2-3人
路人：用剪影/背影/虚化处理
```

**原则：** 主角不超过2人，配角不超过3人

---

### ❌ 错误3：场景切换过于频繁

**错误案例：**
```
第1场：城门外
第2场：城门内
第3场：大堂
第4场：后院
第5场：书房
第6场：卧室
...
```

**问题：**
- 每个场景都要设计+生成
- 工作量爆炸
- 一致性难以保证

**正确做法：**
```
5分钟短剧：3-5个场景即可
用「时间变化」代替「场景切换」：
- 同一场景：早/午/晚/夜
- 用光线和氛围区分
```

**原则：** 场景越少越好，用时间/光线变化丰富画面

## 二、风格层常见错误

### ❌ 错误4：没有定义风格就开干

**错误案例：**
```
直接开始画人物，没有确定是日漫风还是美漫风
结果：镜头1像日漫，镜头2像美漫，镜头3像真人...
风格混乱，看起来像"拼凑"的
```

**正确做法：**
```
第一步：完成风格定义
1. 确定主风格类型（日漫/美漫/CG/仿真人等）
2. 生成风格锚定提示词
3. 准备风格参考图
4. 后续所有生图都使用相同的风格锚定
```

**原则：** 风格定义是保证一致性的第一步，必须先做！

---

### ❌ 错误5：风格锚定提示词不统一

**错误案例：**
```
镜头1：anime style, Japanese animation...
镜头2：realistic photo, cinematic...
镜头3：oil painting style...
```

**问题：**
- 每个镜头风格锚定不同
- 画面风格混乱

**正确做法：**
```
所有镜头使用相同的风格锚定提示词：
风格锚定：anime style, high school slice of life, 
bright and warm colors, Makoto Shinkai inspired...

镜头1：[风格锚定] + 林小雨站在教室门口...
镜头2：[风格锚定] + 她缓缓走进教室...
镜头3：[风格锚定] + 她坐下...
```

**原则：** 风格锚定提示词必须统一，复制粘贴到每张图

## 三、设计层常见错误

### ❌ 错误6：提示词全是中文

**错误案例：**
```
中文提示词：林小雨，16岁，高中女生，穿着白色校服...
英文提示词：（空着不写）
```

**问题：**
- 大多数AI工具对中文理解有限
- 同样的中文描述，每次生成结果差异大

**正确做法：**
```
中文提示词：林小雨，16岁，高中女生，穿着白色校服...
英文提示词：Lin Xiaoyu, 16 years old, high school girl, 
            wearing white school uniform...
```

**原则：** 中英双语提示词，英文为主

---

### ❌ 错误7：负面提示词不写

**错误案例：**
```
只写了正面提示词，没有写负面提示词
结果：生成的图有多余手指、背景混乱...
```

**正确做法：**
```
负面提示词（必填）：
blurry, low quality, deformed, bad anatomy, 
extra limbs, missing limbs, watermark, text, 
bad face, ugly face, bad hands, extra fingers
```

**原则：** 负面提示词是"排除错误"的关键

---

### ❌ 错误8：人物设计过于复杂

**错误案例：**
```
人物：林小雨
外貌：棕色长发扎成双马尾，用红色蝴蝶结发饰，
      左耳戴着珍珠耳环，右手腕戴着银色手链，
      穿着白色衬衫+蓝色格子百褶裙+白色长袜+黑色小皮鞋
```

**问题：**
- 细节太多，AI难以保证一致性
- 每个镜头都要检查所有细节

**正确做法：**
```
人物：林小雨
外貌：棕色双马尾，白色校服，蓝色格子裙
     （抓住3-5个核心特征即可）
```

**原则：** 核心特征3-5个，细节交给负面提示词排除

## 四、分镜层常见错误

### ❌ 错误9：分镜表没有标注时长

**错误案例：**
```
| 镜头 | 画面描述 | 提示词 |
|------|----------|--------|
| 1 | 林小雨站在教室门口 | ... |
| 2 | 她缓缓走进教室 | ... |
```

**问题：**
- 不知道每个镜头应该多长
- 剪辑时节奏把控困难

**正确做法：**
```
| 镜头 | 画面描述 | 时长 | 提示词 |
|------|----------|------|--------|
| 1 | 林小雨站在教室门口 | 3秒 | ... |
| 2 | 她缓缓走进教室 | 5秒 | ... |
```

**原则：** 每个镜头都要标注时长

---

### ❌ 错误10：动作描述太复杂

**错误案例：**
```
动作：林小雨快速跑向门口，推开门，转身，挥手说再见，
      然后跳着离开
```

**问题：**
- 一个镜头包含多个动作
- AI视频难以处理连续复杂动作

**正确做法：**
```
拆分成多个镜头：
镜头1：林小雨跑向门口（3秒）
镜头2：她推开门（2秒）
镜头3：转身挥手说再见（3秒）
镜头4：跳着离开（全景）（4秒）
```

**原则：** 一个镜头一个动作

---

### ❌ 错误11：没有设计备选方案

**错误案例：**
```
只设计了一套分镜方案
生成失败后，不知道该怎么办...
```

**正确做法：**
```
每个关键镜头设计备选方案：
方案A：近景特写（首选）
方案B：中景画面（备选）
方案C：远景画面（保底）
```

**原则：** 关键镜头至少准备2套方案

---

## 五、视频层常见错误

### ❌ 错误12：期望AI生成完美视频

**错误案例：**
```
期望：AI直接生成完美流畅的视频，无需后期
现实：人物变形、动作不自然、画面闪烁...
```

**正确做法：**
```
调整预期：
1. AI视频是"素材"，不是"成品"
2. 需要后期筛选、剪辑、优化
3. 不满意的片段要重新生成或用静态图替代
```

**原则：** AI视频是半成品，后期优化是必须的

---

### ❌ 错误13：忽视音效的作用

**错误案例：**
```
只关注画面，完全忽略声音设计
结果：视频看起来很"干"，没有氛围感
```

**正确做法：**
```
音效设计清单：
- 环境音：雨声、风声、街道嘈杂声...
- 动作音：脚步声、开门声、翻书声...
- 情绪音：心跳声、呼吸声...
- 背景音乐：根据情绪选择
```

**原则：** 好的音效可以弥补画面的不足

## 六、项目管理常见错误

### ❌ 错误14：没有建立资产库

**错误案例：**
```
生成的图片散落在各个文件夹
找不到某个角色的参考图
忘记某个场景的提示词...
```

**正确做法：**
```
建立标准化资产库：
资产库/
├── 人物/
│   ├── 林小雨/
│   │   ├── 设定.md
│   │   ├── 提示词/
│   │   └── 参考图/
├── 场景/
│   ├── 教室/
│   │   ├── 设定.md
│   │   ├── 提示词/
│   │   └── 参考图/
```

**原则：** 所有资产统一管理，便于查找和复用

---

### ❌ 错误15：没有阶段性自检

**错误案例：**
```
一口气做到最后，发现：
- 人物设计有问题
- 分镜逻辑不通
- 视频风格不一致
全部重来...
```

**正确做法：**
```
五阶段自检：
1. 故事层结束 → AI可行性评估
2. 风格层结束 → 风格定义验收
3. 设计层结束 → 资产库验收
4. 分镜层结束 → 分镜图质量检查
5. 视频层结束 → 成片质量验收
```

**原则：** 每个阶段都要验收，问题早发现早解决

## 七、工具使用常见错误

### ❌ 错误16：盲目追求最新工具

**错误案例：**
```
听说HappyHorse很强，就等HappyHorse发布
结果：项目拖延，其他工具也没学会...
```

**正确做法：**
```
1. 先用现有工具跑通流程
2. 新工具开放后再考虑迁移
3. 技能比工具更重要
```

**原则：** 工具是手段，内容才是目的

## 八、避坑检查清单

制作前，对照检查：

### 故事层
- [ ] 主角不超过2人
- [ ] 场景不超过5个
- [ ] 无复杂动作设计
- [ ] 已进行AI可行性评估

### 风格层
- [ ] 已确定主风格类型
- [ ] 已生成风格锚定提示词
- [ ] 已准备风格参考图
- [ ] 风格锚定已准备好用于所有生图

### 设计层
- [ ] 所有提示词都包含风格锚定
- [ ] 提示词中英双语
- [ ] 包含负面提示词
- [ ] 资产库结构完整

### 分镜层
- [ ] 每个镜头标注时长
- [ ] 一个镜头一个动作
- [ ] 关键镜头有备选方案
- [ ] 已建立分镜图库

### 视频层
- [ ] 已设计音效方案
- [ ] 预留后期优化时间
- [ ] 有质量验收标准
