## 【S-001】高二(3)班教室

### 基本信息

| 项目 | 内容 |
|------|------|
| 场景名称 | 高二(3)班教室 |
| 场景类型 | 室内/校园/日常 |
| 时间 | 白天（上午/下午） |
| 光线 | 自然光，窗户阳光 |
| 氛围 | 明亮、青春、日常 |

### 场景描述

**布局：**
- 标准日式高中教室
- 4排×8列课桌
- 李凡座位：靠窗后排
- 黑板在前方，讲台在中间

**关键元素：**
- 窗户（阳光透过）
- 黑板（"新学期，新起点"）
- 课桌椅
- 值日角
- 后门

### 场景参考图关键词

**中文：**
```
日式高中教室，明亮，阳光透过窗户，
课桌椅整齐排列，黑板，讲台，
青春校园氛围，日漫风格。
```

**英文提示词：**
```
Japanese high school classroom,
bright natural lighting, sunlight through windows,
rows of desks and chairs, blackboard, podium,
clean and tidy, school life atmosphere,
anime style, manga style,
slice of life, youthful vibes,
wide angle shot, establishing shot.
```

### 时间变体

| 时间 | 光线变化 | 氛围变化 |
|------|----------|----------|
| 上午 | 明亮，阳光斜射 | 清新、开始 |
| 下午 | 温暖，金色阳光 | 温馨、日常 |
| 傍晚 | 橙色夕阳 | 略带忧伤 |

### 关键镜头位置

1. **教室全景**：展示整体布局
2. **李凡座位特写**：他和苏晴的座位
3. **窗户视角**：阳光洒进来
4. **黑板特写**：开学标语

## 【S-003】学校后门小巷

### 基本信息

| 项目 | 内容 |
|------|------|
| 场景名称 | 学校后门小巷 |
| 场景类型 | 室外/战斗/紧张 |
| 时间 | 傍晚 |
| 光线 | 昏暗、戏剧化 |
| 氛围 | 紧张、危险、战斗 |

### 场景描述

**布局：**
- 狭窄的小巷
- 两侧是老旧建筑
- 地面有杂物
- 尽头是学校围墙

**关键元素：**
- 老旧砖墙
- 电线杆
- 垃圾桶
- 阴影角落

### 战斗状态变化

| 状态 | 光线 | 氛围 |
|------|------|------|
| 战斗前 | 昏暗、压抑 | 不安、威胁 |
| 战斗中 | 特效光（金/蓝/紫） | 紧张、激烈 |
| 战斗后 | 夕阳余晖 | 平静、余韵 |

### 英文提示词

**战斗前（阴暗）：**
```
narrow alleyway behind school,
old brick walls, power lines above,
dim lighting, ominous atmosphere,
shadows in corners, dangerous vibes,
anime style, manga style,
dramatic dark lighting, tense mood.
```

**战斗中（特效）：**
```
narrow alleyway, battle scene,
golden aura vs ice blue vs dark purple,
dramatic lighting, intense action,
magical effects, power clash,
anime style, manga style,
dynamic composition, epic battle.
```

**战斗后（夕阳）：**
```
narrow alleyway at sunset,
golden hour lighting, warm orange sky,
peaceful after battle atmosphere,
anime style, manga style,
beautiful sunset, emotional mood.
```

### 风格锚定提示词

每个场景都添加以下风格词：

```
anime background, Japanese anime style,
detailed environment, atmospheric lighting,
《获得异能的那一天，我和校花成为了同桌》官方场景风格
```

### 场景连续性

- S-003 和 S-004 是同一地点不同时间
- 保持建筑结构一致
- 只改变光线和氛围
