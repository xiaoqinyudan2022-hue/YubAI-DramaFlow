## 【李凡】- 男主角

### 基本信息

| 项目 | 内容 |
|------|------|
| 年龄 | 17岁 |
| 身高 | 175cm |
| 异能 | 感知系（能看到"气"） |
| 性格 | 普通、善良、有点怂但关键时刻靠谱 |
| 觉醒特征 | 眼睛发出金色光芒 |

### 外貌描述

**日常状态：**
- 黑色短发，有点凌乱
- 普通长相，不算帅但有辨识度
- 标准校服，白衬衫+深蓝外套
- 表情：紧张、困惑、普通高中生

**觉醒状态：**
- 瞳孔变成金色
- 身边泛着淡淡的金色光芒
- 表情：坚定、觉醒后的决心

### 核心特征词

**中文：**
```
李凡，17岁男生，黑色短发，普通长相，
穿校服白衬衫，身高175cm，
表情紧张但眼神善良。
日漫风格，少年感。
```

**英文提示词：**
```
Li Fan, 17-year-old Japanese anime boy,
black short messy hair, average appearance,
wearing white school uniform shirt with blue jacket,
normal height 175cm, 
nervous expression but kind eyes,
anime style, manga style, 
dramatic lighting, highly detailed.
```

**觉醒状态提示词：**
```
Li Fan, 17-year-old Japanese anime boy,
black short hair, glowing golden eyes,
golden aura surrounding him,
determined expression, awakened power,
anime style, manga style,
dramatic golden lighting, dynamic pose,
powerful atmosphere.
```

### 角色表情参考

| 表情 | 描述 | 适用场景 |
|------|------|----------|
| 紧张 | 眉头紧皱，冷汗 | 和苏晴同桌、危机时 |
| 懵逼 | 嘴巴微张，眼神呆滞 | 被告知成为搭档 |
| 觉醒坚定 | 眼睛发光，表情坚毅 | 关键时刻爆发 |
| 日常普通 | 无表情或轻微困惑 | 日常场景 |

### 服装设定

**校服（日常）：**
- 白色衬衫
- 深蓝色西装外套
- 深蓝色校裤
- 普通运动鞋

## 生成建议

### 文生图优先级

1. **GPT-Image-2**：提示词理解最强，适合复杂场景
2. **NanoBanana Pro**：照片级真实感，适合人物立绘
3. **Midjourney**：艺术感强，适合氛围图

### 一致性控制

**方法1：风格锚定**
- 每张图都加上相同的核心风格词
- `anime style, manga style, Japanese anime`

**方法2：参考图控制**
- 先生成满意的人物立绘
- 后续生成时用参考图保持一致

**方法3：特征词固定**
- 李凡：black short messy hair, golden eyes (awakened)
- 苏晴：long straight black hair, ice princess vibe
- 每次生成都使用相同的特征描述
