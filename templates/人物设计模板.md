## AI提示词库

### 0. 风格锚定（必填）

> ⚠️ 重要：风格锚定保证所有人物生成风格一致，请在项目开始时定义，所有人物共用！

**风格锚定提示词：**
```
[风格类型]，[色调偏好]，[画面质感]，[参考风格]...
例如：anime style, bright warm colors, soft lighting,
      Makoto Shinkai inspired, clean line art, masterpiece
```

---

### 1. 基础提示词（核心特征）

**中文提示词：**
```
[风格锚定]
[角色名]，[年龄段]，[身份]。
[脸型描述]，[眼睛描述]，[发型描述]，[肤色描述]。
[体型描述]，[气质描述]。
高质量。
```

**英文提示词：**
```
[Style Anchor]
[Name], [age] years old, [role/identity].
[face description], [eye description], [hair description], [skin description].
[body type], [temperament].
masterpiece, best quality, 8K.
```

**负面提示词：**
```
blurry, low quality, deformed, bad anatomy, extra limbs, missing limbs,
watermark, text, signature, bad face, ugly face, mutation, disfigured
```
