## 剧本格式规范

### 场景标题格式

```
【场景编号】
内/外 · 场景地点 · 时间
```

**示例：**
```
【第1场】
内 · 林小雨的房间 · 夜
```

### 角色标注

```
角色名：（动作/表情）"台词"
```

**示例：**
```
林小雨：（轻轻抬起头，眼眶微红）"我真的能做到吗？"
张老师：（微微一笑）"相信自己，你一定可以。"
```

### 旁白格式

```
（旁白）：[旁白内容]
```

### 镜头指示（可选）

```
【镜头1】：特写，林小雨的手在画纸上移动
【镜头2】：中景，林小雨坐在窗边
```
