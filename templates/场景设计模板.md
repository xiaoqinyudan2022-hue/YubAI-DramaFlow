## AI提示词库

### 0. 风格锚定（必填）

> ⚠️ 重要：风格锚定保证所有场景生成风格一致，请在项目开始时定义，所有场景共用！

**风格锚定提示词：**
```
[风格类型]，[色调偏好]，[画面质感]，[参考风格]...
例如：anime style, bright warm colors, soft lighting,
      Makoto Shinkai inspired, clean line art, masterpiece
```

---

### 1. 基础场景提示词

**中文提示词：**
```
[风格锚定]
[场景名称]，[场景类型]。
[详细描述：建筑风格、室内布置、重要元素]。
[光线描述]，[时间]，[天气]。
[色调描述]，[氛围]。
```

**英文提示词：**
```
[Style Anchor]
[Scene name], [scene type].
[Detailed description: architecture style, interior, key elements].
[Lighting description], [time], [weather].
[Color tone], [atmosphere].
highly detailed, 8K, masterpiece.
```

**负面提示词：**
```
blurry, low quality, deformed, watermark, text, signature,
ugly, disfigured, bad architecture, distorted building
```
